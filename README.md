# ⚡ SHADOW FLAMEZ AI STUDIO PRO v2.0

> An interactive, high-performance web suite built for AI background removal, compositing, neural super-resolution, batch operations, and image color grading.

---

## 🔥 Key Features

* 🌌 **Background & Compositor:** AI cutout generation, solid color replacements, custom background overlays, and drop-shadow sliders.
* 🚀 **4x AI Super-Resolution:** Upscale low-res graphics with neural clarity and pixel matching.
* 🏷️ **Watermark & Branding:** Precise logo placement, scaling, position anchoring, and opacity customization.
* ⚡ **1-Click Pipeline Transfers:** Seamlessly push rendered images across tools (Compositor → Upscaler → Watermark).
* 🎨 **Color & Tone Studio:** Pre-built cyberpunk color presets, brightness, contrast, saturation, and sharpness tuning.
* 📦 **Batch Processing Matrix:** Process multi-image queues simultaneously with transparent alpha cutout exports.
* 📽️ **Smart Canvas & Export:** Trim transparent whitespace, fit aspect ratios (1:1, 16:9, 9:16), and export in PNG, JPEG, or WEBP.

---

## 🛠️ Tech Stack

* **UI Framework:** [Gradio](https://gradio.app/)
* **Language:** Python 3.10+
* **Styling:** Custom Glassmorphism & Cyberpunk CSS
* **Deployment:** Hosted on Cloud Infrastructure
